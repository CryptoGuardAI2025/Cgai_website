
# CGAI High-End Website

Diese Version ist optimiert für GitHub Pages.

## Deployment Anleitung

1. Repository auf GitHub erstellen (Name: cgai_highend oder beliebig).
2. Alle Dateien und Ordner aus diesem ZIP hochladen (root-Ebene).
3. In den Repository-Einstellungen (Settings) -> Pages -> Branch auf "main" setzen und Ordner "/" auswählen.
4. Speichern. Nach wenigen Minuten ist die Seite online unter: https://DEIN_GITHUB_NAME.github.io/REPO_NAME/

## Inhalte anpassen
- Texte: HTML-Dateien direkt auf GitHub bearbeiten.
- Logo: `assets/logo.svg` ersetzen.
- PDFs: Neue Dateien in `pdfs` hochladen (Dateiname gleich lassen).
